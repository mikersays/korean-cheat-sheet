00 Foundations · 기초

Cushion vowels and 받침 — does the syllable end in a consonant? → read the rule.

0.1identify
Does 학교 end in a 받침?
답
No — 교 ends in a vowel (ㅛ). The block has no final consonant.
0.2identify
Does 책 end in a 받침?
답
Yes — ㄱ is the 받침 of 책.
0.3choose
Pick the correct cushion-vowel form: 먹 + (으)면 →
  • 먹면
  • 먹으면
  • 머그면
답
먹으면 — the stem ends in a consonant (받침 ㄱ), so insert 으.
0.4choose
Pick the correct cushion-vowel form: 가 + (으)면 →
  • 가면
  • 가으면
답
가면 — the stem ends in a vowel, so the cushion 으 is dropped.

01 Word Order — SOV

Subject — object — verb. Verb always last. → read the rule.

1.1write
Translate to Korean (SOV order): "I eat a book" — (silly on purpose; just check word order) Use 저는, 책을, 먹어요.
답
저는 책을 먹어요. — Subject + Object + Verb. Verb always last.
1.2identify
In 저는 물을 마셔요, which word carries the action? (I drink water.)
답
마셔요 — it is the verb, and in Korean the verb always comes last.
1.3transform
Reorder into correct Korean SOV: 읽어요 / 저는 / 책을
답
저는 책을 읽어요. — Subject (저는) + Object (책을) + Verb (읽어요).
1.4write
Translate to Korean (SOV order): "The student studies Korean." Use 학생은, 한국어를, 공부해요.
답
학생은 한국어를 공부해요. — The subject 학생은 comes first; the verb 공부해요 comes last.
1.5read
Read 친구는 영화를 봐요 and identify the subject, object, and verb. (My friend watches a movie.)
답
Subject: 친구는 — Object: 영화를 — Verb: 봐요. This is the standard Subject–Object–Verb order.
1.6choose
Which sentence has the correct Korean SOV word order?
  • 저는 마셔요 커피를.
  • 저는 커피를 마셔요.
  • 마셔요 저는 커피를.
답
저는 커피를 마셔요. — Subject first, object in the middle, verb last; the verb must close the sentence.

02 Particles 조사

Tiny suffixes that mark each noun's job in the sentence. → read the rule.

2.1particle
Fill in the blank: 저___ 학생이에요. (I'm a student.)
답
는 — topic particle. 저 ends in a vowel, so 는 not 은.
2.2choose
Which particle? 학교___ 가요.
  • 에
  • 에서
(I'm going to school.)
답
에 — destination with a motion verb. 에서 would mark the location where something happens, not the destination.
2.3particle
Fill in the blank: 친구___ 만나요. (I'm meeting a friend.)
답
를 — direct object. 친구 ends in a vowel, so 를 not 을.
2.4particle
Fill in the blank: 도서관___ 공부해요. (I study at the library.)
답
에서 — action location particle; the studying happens at the library, so 에서 not 에.
2.5choose
Which particle? 책___ 읽어요.
  • 을
  • 를
(I read a book.)
답
을 — direct object particle. 책 ends in a consonant (받침 ㄱ), so 을 not 를.
2.6particle
Fill in the blank: 선생님___ 학교에 있어요. (The teacher is at school. — use the subject particle)
답
이 — subject particle. 선생님 ends in a consonant (받침 ㅁ), so 이 not 가. Use 이/가 (vs topic 은/는) when introducing new information or describing a state.
2.7choose
Which "and/with" particle works in 친구___ 밥을 먹어요.?
  • 와
  • 과
  • 하고
(I eat with a friend.)
답
와 or 하고 both work. 친구 ends in a vowel, so 와 (not 과). 하고 is the spoken/casual equivalent — no 받침 distinction, attaches to any noun.
2.8read
Read 저도 커피를 마셔요 and explain what 도 does here. (I drink coffee too.)
답
도 means "also / too" and replaces the topic or subject particle — it cannot be stacked on top of 은/는 or 이/가.
2.9particle
Fill in the blank: 물___ 마셔요. (I drink only water.)
답
만 — "only" particle; 물만 마셔요 means "I drink only water".
2.10choose
Which particle? 젓가락___ 먹어요.
  • 으로
  • 로
(I eat with chopsticks.)
답
으로 — means-of instrument particle. 젓가락 ends in a consonant (받침 ㄱ), so 으로 not 로.
2.11read
Read 9시부터 6시까지 일해요 and explain what 부터 and 까지 mark. (I work from 9 to 6.)
답
부터 marks the starting time ("from") and 까지 marks the end point ("to/until") — used together for a time range.
2.12particle
Fill in the blank: 친구___ 집에 가요. (I'm going to a friend's house. — use possessive "of")
답
의 — possessive particle meaning "of / 's"; 친구의 집 = "friend's house".
2.13read
Read 학교에서 한국어를 배워요 and name the function of each particle. (I learn Korean at school.)
답
에서 marks the location where learning happens; 를 marks 한국어 as the direct object of the verb.
2.14write
Write in Korean: "I buy a book at the store." Use 저는, 가게에서, 책을, 사요.
답
저는 가게에서 책을 사요. — 에서 marks where the buying action takes place; 을 marks the direct object.
2.15identify
This sentence has a particle error — what is it? 저는 밥는 먹어요. (I eat rice.)
답
밥는 is wrong — 밥 is the direct object, so it needs 을 (consonant-final), not the topic particle 는. Correct: 저는 밥을 먹어요.

03 The Copula 이다 / 아니다

이다 attaches to nouns to mean 'is/am/are'; negate with 이/가 아니다. → read the rule.

3.1choose
Which copula ending is correct? 저는 의사___.
  • 이에요
  • 예요
(I am a doctor.)
답
예요 — 의사 ends in a vowel, so the cushion 이 is dropped: 의사예요.
3.2choose
Which copula ending is correct? 저는 학생___.
  • 이에요
  • 예요
(I am a student.)
답
이에요 — 학생 ends in a consonant (받침 ㅇ), so the cushion 이 is kept.
3.3transform
Negate the sentence: 저는 선생님이에요. (I am a teacher.)
답
저는 선생님이 아니에요. — Negation uses 이/가 아니에요; the noun takes the subject particle before 아니에요.
3.4transform
Convert to formal 합니다체: 저는 학생이에요. (I am a student.)
답
저는 학생입니다. — 이에요/예요 is polite informal (해요체); 입니다 is formal (합니다체).
3.5identify
Is this copula usage correct? 저는 친구 이에요. (written with a space before 이에요) (I am a friend.)
답
No — 이에요 attaches directly to the noun with no space: 저는 친구예요 (since 친구 ends in a vowel).
3.6conjugate
Give the polite informal copula form for: 사람 (person) Complete: 저는 사람___.
답
저는 사람이에요. — 사람 ends in a consonant (받침 ㅁ), so attach 이에요 not 예요.

04 Verb / Adjective Stems

Drop 다 from a dictionary form to get the stem; everything hooks onto the stem. → read the rule.

4.1transform
Extract the stem: 가다
답
Stem: 가- — drop 다; the stem ends in a vowel, so no cushion is needed for vowel-starting endings.
4.2transform
Extract the stem: 먹다
답
Stem: 먹- — drop 다; the stem ends in a consonant (받침 ㄱ), so consonant-sensitive endings need a cushion 으.
4.3transform
Extract the stems of these three verbs: 보다, 읽다, 공부하다
답
Stems: 보- (vowel-final), 읽- (consonant-final, receives cushion), 공부하- (vowel-final).
4.4identify
Classify each verb as action (동사) or descriptive (형용사): 예쁘다, 오다, 크다, 마시다
답
예쁘다 (to be pretty) and 크다 (to be big) are descriptive verbs; 오다 (to come) and 마시다 (to drink) are action verbs.
4.5identify
Does the stem of 듣다 end in a consonant or a vowel?
답
The stem is 듣-, which ends in a consonant (받침 ㄷ), so endings with a cushion slot will need 으.
4.6identify
Classify each verb as action (동사) or descriptive (형용사): 좋다, 쓰다, 비싸다, 일하다
답
좋다 (to be good) and 비싸다 (to be expensive) are descriptive verbs; 쓰다 (to write) and 일하다 (to work) are action verbs.

05 Speech Levels

해요체 (polite), 합니다체 (formal), 해체/반말 (casual). Pick by audience and context. → read the rule.

5.1identify
What speech level is 먹어요?
답
해요체 (polite) — the ~아요/어요 ending marks this as everyday polite speech, suitable for most situations with strangers or acquaintances.
5.2transform
Convert to formal 합니다체: 가요. (I go / I'm going.)
답
갑니다. — the stem 가- ends in a vowel, so attach ㅂ니다 (not 습니다) to get the formal declarative.
5.3transform
Convert to casual 반말: 먹어요. (I eat / I'm eating.)
답
먹어. — drop 요 from the 해요체 form to get casual 반말, used with close friends, family, or younger people.
5.4choose
You are writing a formal business email to a client you have never met. Which speech level should you use?
  • 해체 / 반말 (먹어, 가)
  • 해요체 (먹어요, 가요)
  • 합니다체 (먹습니다, 갑니다)
답
합니다체 — the formal level is expected in business writing, news, and formal contexts; 해요체 is polite but informal for professional correspondence.
5.5identify
What speech level is 공부합니다?
답
합니다체 (formal-polite) — the ~ㅂ니다/습니다 ending is used in formal situations like news, official announcements, and business.
5.6choose
You are talking with a close friend of the same age. Which speech level is most natural?
  • 합니다체 (갑니다)
  • 해요체 (가요)
  • 해체 / 반말 (가)
답
해체 / 반말 — casual speech is natural with close friends of the same age; using 합니다체 or even 해요체 would sound stiff and distant.

06 Present Tense — 아/어/해요

Stem + 아/어요, with vowel harmony and contraction. → read the rule.

6.1conjugate
Conjugate 먹다 in 해요-form, present tense.
답
먹어요 — stem 먹 has a dark vowel (ㅓ), so it takes 어요.
6.2conjugate
Conjugate 가다 in 해요-form, present tense.
답
가요 — stem 가 ends in ㅏ (bright vowel), so it takes 아요; 가 + 아요 contracts to 가요 (same vowel merges).
6.3conjugate
Conjugate 보다 in 해요-form, present tense.
답
봐요 — stem 보 ends in ㅗ (bright vowel), so it takes 아요; ㅗ + ㅏ contracts to ㅘ, giving 봐요.
6.4conjugate
Conjugate 마시다 in 해요-form, present tense.
답
마셔요 — stem 마시 ends in ㅣ (neutral; not ㅏ or ㅗ), so it takes 어요; ㅣ + ㅓ contracts to ㅕ, giving 마셔요.
6.5conjugate
Conjugate 자다 in 해요-form, present tense.
답
자요 — stem 자 ends in ㅏ (bright vowel), so it takes 아요; 자 + 아요 contracts to 자요.
6.6conjugate
Conjugate 사다 in 해요-form, present tense.
답
사요 — stem 사 ends in ㅏ (bright vowel), so it takes 아요; 사 + 아요 contracts to 사요.
6.7conjugate
Conjugate 읽다 in 해요-form, present tense.
답
읽어요 — stem 읽 ends in a consonant cluster (받침 ㄱ), last vowel is ㅣ (not ㅏ or ㅗ), so it takes 어요; no contraction since the stem ends in a consonant.
6.8conjugate
Conjugate 살다 in 해요-form, present tense.
답
살아요 — stem 살 has a bright vowel (ㅏ) and ends in 받침 ㄹ, so it takes 아요 with no vowel contraction: 살아요.
6.9conjugate
Conjugate the 하다 verb 공부하다 in 해요-form, present tense.
답
공부해요 — all 하다 verbs contract to 해요 in the present polite form: 하 + 아요 → 해요.
6.10conjugate
Conjugate the 하다 verb 일하다 in 해요-form, present tense.
답
일해요 — 하다 always contracts to 해요: 일하 + 아요 → 일해요.
6.11choose
Which 해요-form is correct for 보다?
  • 보어요
  • 봐요
답
봐요 — when the stem vowel is ㅗ and the ending starts with ㅏ, they contract to ㅘ: 보 + 아요 → 봐요. The uncontracted 보어요 is not used.
6.12identify
What is the dictionary form of the verb 입어요?
답
입다 — the 해요-form 입어요 comes from stem 입- (consonant-final, vowel ㅣ takes 어요); restore the stem and add 다 → 입다 (to wear).

07 Past Tense — 았/었/했어요

Past = 해요-form with 어요 → 었어요. → read the rule.

7.1transform
Change to past tense: 가요 (goes / is going)
답
갔어요 — 해요-form present is 가요; for past, 가 + 아요 → 가요 becomes 가 + 았어요 → 갔어요 (ㅏ + ㅏ contracts, adding past marker).
7.2transform
Change to past tense: 먹어요 (eats)
답
먹었어요 — stem 먹 takes 었 (dark vowel); 먹 + 었어요 → 먹었어요.
7.3transform
Change to past tense: 봐요 (watches / sees)
답
봤어요 — stem 보 takes 았 (bright vowel ㅗ); 보 + 았어요 → 봤어요 (same ㅗ+ㅏ → ㅘ contraction).
7.4transform
Change to past tense: 마셔요 (drinks)
답
마셨어요 — stem 마시 takes 었 (vowel ㅣ is not ㅏ/ㅗ); 마시 + 었어요 → 마셨어요 (ㅣ + ㅓ → ㅕ contraction).
7.5transform
Change to past tense: 자요 (sleeps)
답
잤어요 — stem 자 takes 았 (bright vowel ㅏ); 자 + 았어요 → 잤어요.
7.6conjugate
Conjugate 좋다 in past tense (해요-form).
답
좋았어요 — stem 좋 has a bright vowel (ㅗ), so it takes 았어요: 좋 + 았어요 → 좋았어요.
7.7conjugate
Conjugate 예쁘다 in past tense (해요-form). (Caution: this is an irregular verb.)
답
예뻤어요 — 예쁘다 is a ㅡ-irregular: the stem vowel ㅡ drops before a vowel-starting ending. Look back to the previous syllable's vowel to pick harmony — here 예 (ㅖ, neutral) → use 었어요: 예쁘 + 었어요 → 예뻤어요.
7.8conjugate
Conjugate 사다 in past tense (해요-form).
답
샀어요 — stem 사 ends in bright vowel ㅏ, so it takes 았어요; 사 + 았어요 → 샀어요.
7.9conjugate
Conjugate the 하다 verb 공부하다 in past tense (해요-form).
답
공부했어요 — all 하다 verbs use 했어요 in past tense: 하 + 았어요 → 했어요.
7.10identify
What is the dictionary form of the verb in this past-tense sentence? 어제 친구를 만났어요. (I met a friend yesterday.)
답
만나다 — past form 만났어요 comes from stem 만나 + 았어요 (bright vowel ㅏ), contracted; dictionary form is 만나다 (to meet).

08 Future / Probability

~(으)ㄹ 거예요 for plain future; ~겠어요 for intent or supposition. → read the rule.

8.1conjugate
Conjugate 가다 in the plain future form ~(으)ㄹ 거예요.
답
갈 거예요 — stem 가 ends in a vowel (no 받침), so attach ㄹ 거예요 directly: 가 + ㄹ 거예요 → 갈 거예요.
8.2conjugate
Conjugate 먹다 in the plain future form ~(으)ㄹ 거예요.
답
먹을 거예요 — stem 먹 ends in a consonant (받침 ㄱ), so the cushion 으 is inserted: 먹 + 을 거예요 → 먹을 거예요.
8.3conjugate
Conjugate 보다 in the plain future form ~(으)ㄹ 거예요.
답
볼 거예요 — stem 보 ends in a vowel (no 받침), so attach ㄹ 거예요 directly: 보 + ㄹ 거예요 → 볼 거예요.
8.4conjugate
Conjugate 사다 (to buy) in the plain future form ~(으)ㄹ 거예요. Note: the answer happens to be a homograph of the future of 살다 (to live) — context disambiguates.
답
살 거예요 — stem 사 ends in a vowel (no 받침), so attach ㄹ 거예요 directly: 사 + ㄹ 거예요 → 살 거예요.
8.5transform
Convert to future tense using ~(으)ㄹ 거예요: 먹어요 (eats)
답
먹을 거예요 — return to the stem 먹 (consonant-final) and add 을 거예요.
8.6transform
Convert to future tense using ~(으)ㄹ 거예요: 공부해요 (studies)
답
공부할 거예요 — stem 공부하 ends in a vowel, so attach ㄹ 거예요: 공부하 + ㄹ 거예요 → 공부할 거예요.
8.7read
Read this sentence and explain what ~겠어요 signals here: 비가 오겠어요.
답
비가 오겠어요 means "It will probably rain" or "It looks like it will rain." The ~겠~ form conveys supposition or inference based on evidence (e.g., looking at the sky) — not a simple future statement. Compare: 비가 올 거예요 is a neutral plain future prediction.
8.8choose
Which form best expresses "I will do it (strong personal intent/commitment)"?
  • 제가 할 거예요.
  • 제가 하겠습니다.
답
제가 하겠습니다 — ~겠~ signals strong personal intent or formal commitment; ~(으)ㄹ 거예요 is a neutral plain future. Both are grammatical, but 하겠습니다 is the standard way to volunteer or commit.

09 Negation

안 (won't/don't), 못 (can't); 하다 verbs split: 안 X-해요 → X 안 해요. → read the rule.

9.1transform
Negate with short-form 안: 먹어요 (eats)
답
안 먹어요 — place 안 directly before the verb: 안 먹어요 (doesn't eat / won't eat).
9.2transform
Negate with short-form 안: 가요 (goes)
답
안 가요 — place 안 directly before the verb: 안 가요 (doesn't go / won't go).
9.3transform
Negate with short-form 안, correctly splitting the 하다 verb: 공부해요 (studies)
답
공부 안 해요 — with 하다 compound verbs, 안 splits between the noun and 해요: 공부 안 해요. Saying 안 공부해요 is non-standard.
9.4transform
Negate with short-form 안, correctly splitting the 하다 verb: 일해요 (works)
답
일 안 해요 — 일하다 is a noun-하다 compound; 안 inserts between the noun 일 and 해요.
9.5choose
Which is the correct negation for "I don't drink coffee" (by choice)?
  • 저는 커피를 안 마셔요.
  • 저는 커피를 못 마셔요.
답
저는 커피를 안 마셔요 — 안 expresses unwillingness or habitual non-action (choice); 못 expresses inability or physical/circumstantial impossibility. "I don't drink coffee" by preference uses 안.
9.6choose
Which is the correct negation for "I can't drink coffee" (e.g., due to an allergy)?
  • 저는 커피를 안 마셔요.
  • 저는 커피를 못 마셔요.
답
저는 커피를 못 마셔요 — 못 expresses inability (physical or circumstantial); use it when some outside factor prevents the action, not when it is merely a personal choice.
9.7transform
Negate using the long form ~지 않다: 먹어요 (eats)
답
먹지 않아요 — long-form negation: stem 먹 + 지 않아요. This is interchangeable with 안 먹어요 but slightly more emphatic or formal in writing.
9.8identify
Is this negation correct? 저는 안 청소해요. (I don't clean.)
답
Non-standard — 청소하다 is a noun-하다 compound, so 안 should split between noun and verb: 저는 청소 안 해요. The form 안 청소해요 is heard in fast casual speech but is considered non-standard.

10 Honorifics — ~(으)시~

Insert ~(으)시~ when the subject is socially elevated; some verbs have suppletive forms. → read the rule.

10.1transform
Add the honorific ~(으)시~ to make this sentence respectful. Subject is your teacher. 가요. (goes)
답
가세요 — stem 가 ends in a vowel, so attach 시 directly: 가 + 시 + 어요 → 가세요 (시+어 contracts to 세).
10.2identify
The verb 먹다 has a suppletive honorific form. What is the polite honorific present-tense form when the subject is your professor eating lunch?
답
드세요 — 먹다 (to eat) has the suppletive honorific 드시다; present polite form is 드시 + 어요 → 드세요. Do not use 먹으세요 for the subject's eating.
10.3transform
Add the honorific ~(으)시~: 와요 (the subject is your grandmother coming home). (comes)
답
오세요 — stem 오 ends in a vowel; 오 + 시 + 어요 → 오세요. Note: 오다 does not have a suppletive form; the regular ~(으)시~ insertion applies.
10.4transform
Convert to honorific past tense: 갔어요 (the subject is your teacher). (went)
답
가셨어요 — insert 시 into the past form: stem 가 + 시 + 었어요 → 가셨어요 (시+었 → 셨).
10.5transform
Convert to honorific past tense: 했어요 (the subject is your manager). (did)
답
하셨어요 — insert 시: stem 하 + 시 + 었어요 → 하셨어요.
10.6identify
Match the suppletive honorific form: which verb replaces 자다 (to sleep) when the subject is honored?
답
주무시다 — 자다 (to sleep) becomes 주무시다 in the honorific. Present polite form: 주무세요.
10.7identify
Match the suppletive honorific form: which verb replaces 있다 (to be / to exist) when the subject is honored and physically present somewhere?
답
계시다 — 있다 (to be present) becomes 계시다 for an honored subject. Present polite: 계세요. Note: 있다 meaning "to have" uses 있으세요 or the context makes it clear.
10.8read
Should this sentence use honorifics? The subject is your younger sibling. 동생이 자요. (My younger sibling sleeps.)
답
No — honorifics (~(으)시~) are used to honor the subject of the sentence. A younger sibling is not socially elevated relative to the speaker, so plain 자요 is correct. Compare: if the subject were your grandmother, you would say 할머니께서 주무세요.

11 Connectors (clause linking)

Clause-joining endings: 고, 서, 지만, (으)면, (으)니까, 는데, (으)러. → read the rule.

11.1transform
Join these two clauses with ~고 (listing): 밥을 먹다 + 커피를 마시다 (eat rice / drink coffee)
답
밥을 먹고 커피를 마셔요 — ~고 attaches directly to the stem regardless of vowel or consonant ending: 먹 + 고. The second clause takes the polite ending.
11.2transform
Join with ~아/어서 (sequence / cause): 학교에 가다 + 공부하다 (go to school / study)
답
학교에 가서 공부해요 — stem 가 ends in ㅏ, so attach 아서: 가 + 아서 → 가서 (contracts). This is sequential: "go to school and (there) study."
11.3transform
Join with ~아/어서 (reason): 배가 고프다 + 밥을 먹다 (be hungry / eat)
답
배가 고파서 밥을 먹어요 — stem 고프 (ㅡ-irregular): ㅡ drops before 아/어, giving 고파서. This is causal: "because I'm hungry, I eat." Note: ~아/어서 cannot be followed by a command or suggestion.
11.4transform
Join with ~지만 (but, contrast): 비싸다 + 맛있다 (expensive / delicious)
답
비싸지만 맛있어요 — ~지만 attaches to the stem directly: 비싸 + 지만. No tense or vowel-harmony adjustment needed.
11.5transform
Join with ~(으)면 (if): 시간이 있다 + 만나다 (have time / meet)
답
시간이 있으면 만나요 — stem 있 ends in a consonant, so the cushion 으 is added: 있 + 으면. Compare vowel-final stem: 가 + 면 → 가면 (no cushion needed).
11.6transform
Join with ~(으)니까 (because/since): 비가 오다 + 우산을 가져가다 (rain is falling / take an umbrella)
답
비가 오니까 우산을 가져가요 — stem 오 ends in a vowel; attach 니까 directly. ~(으)니까 is used when the second clause is a command or suggestion, which ~아/어서 cannot do.
11.7choose
Which connector is correct here? "It's cold, so wear a coat (please)."
  • 추워서 코트 입으세요.
  • 추우니까 코트 입으세요.
답
추우니까 코트 입으세요 — when the main clause is a command or suggestion (입으세요), ~아/어서 is blocked; you must use ~(으)니까. 추워서 코트 입으세요 is ungrammatical in standard Korean.
11.8transform
Join with ~(으)러 (in order to, motion verb): 밥을 먹다 + 가다 (eat / go)
답
밥을 먹으러 가요 — stem 먹 ends in a consonant, so cushion 으 is added: 먹 + 으러. The main verb must be a motion verb (가다, 오다, 다니다). Compare vowel-final: 사 + 러 → 사러.
11.9identify
What does ~는데 signal in this sentence? 밥을 먹는데 전화가 왔어요. (I was eating when a call came in.)
답
Background context — ~는데 here sets up the situation in the first clause as background for the second event. The four uses of ~는데: (1) contrast ("but"), (2) background ("I was doing X when…"), (3) soft observation (~는데요), (4) lead-in to a question.
11.10choose
Which connector best fills the blank? "I know it's expensive, ___ I'll buy it anyway." (pure contrast, no background setting)
  • 비싸지만 살 거예요.
  • 비싼데 살 거예요.
답
Both are grammatical, but 비싸지만 살 거예요 is the cleaner contrast ("but"). 비싼데 살 거예요 adds a nuance of background / soft leading remark. For a direct "but" contrast with no framing, ~지만 is the unmarked choice.

12 Modifiers — verbs/adj as noun-modifiers

Endings that turn a verb or adjective into a noun-modifier (relative clause): ~(으)ㄴ / ~는 / ~(으)ㄹ. → read the rule.

12.1transform
Add the past modifier to the action verb: 읽다 + 책 (read / book → "the book I read")
답
읽은 책 — past action-verb modifier is ~(으)ㄴ: stem 읽 ends in a consonant, so add 은: 읽 + 은 → 읽은.
12.2transform
Add the present modifier to the action verb: 읽다 + 책 (read / book → "the book I'm reading")
답
읽는 책 — present action-verb modifier is ~는: attach 는 directly to the stem: 읽 + 는 → 읽는. No vowel harmony or cushion applies here.
12.3transform
Add the future modifier to the action verb: 읽다 + 책 (read / book → "the book I will read")
답
읽을 책 — future modifier is ~(으)ㄹ: stem 읽 ends in a consonant, so add 을: 읽 + 을 → 읽을. Compare vowel-final: 볼 책 (보 + ㄹ).
12.4choose
Which modifier form does a descriptive verb (adjective) use for its present-tense meaning? Choose the correct form for 예쁘다 + 꽃 (pretty flower).
  • 예쁘는 꽃
  • 예쁜 꽃
답
예쁜 꽃 — descriptive verbs use ~(으)ㄴ for their present modifier, not ~는. Stem 예쁘: ㅡ drops before the ending → 예쁘 + ㄴ → 예쁜. Only action verbs use ~는 for the present.
12.5identify
What time does the modifier express? 어제 만난 친구 (the friend [I] met yesterday)
답
Past — ~(으)ㄴ on an action verb marks a completed/past event. 만나 + ㄴ → 만난 (vowel-final stem: ㄴ attaches directly). The modifier describes something that already happened.
12.6read
Translate the modifier phrase: 내일 볼 영화
답
"The movie (I/we) will watch tomorrow" — 볼 is the future modifier: 보 (vowel-final) + ㄹ → 볼. The noun follows the modifier directly, just like an English relative clause but preceding the noun.
12.7transform
Add the present modifier: 재미있다 + 영화 (interesting / movie → "an interesting movie")
답
재미있는 영화 — 있다 and 없다 behave like action verbs for modifiers; they use ~는 for present: 재미있 + 는 → 재미있는.
12.8choose
Which form correctly expresses "the coffee I drank this morning"?
  • 오늘 아침에 마시는 커피
  • 오늘 아침에 마신 커피
답
오늘 아침에 마신 커피 — past action-verb modifier ~(으)ㄴ: 마시 + ㄴ → 마신. The event (drinking) is completed, so past modifier is needed. 마시는 커피 would mean coffee currently being drunk.
12.9transform
Build the noun phrase: 가다 (future) + 곳 (go / place → "a place to go / the place I'll go")
답
갈 곳 — stem 가 (vowel-final) + future modifier ㄹ attaches as final consonant: 가 + ㄹ → 갈. The phrase 갈 곳이 없어요 (I have nowhere to go) is a common pattern.
12.10read
Identify the time reference for each modifier: 어제 본 영화, 지금 보는 영화, 내일 볼 영화
답
본 → past (보 + ㄴ, action verb past modifier); 보는 → present (보 + 는); 볼 → future (보 + ㄹ). The time adverbs (어제/지금/내일) confirm each respectively.

13 Nominalization (verb → noun)

~기 (gerund), ~(으)ㅁ (formal), ~는 것 (the act of). → read the rule.

13.1transform
Nominalize with ~기: 수영하다 (to swim → "swimming")
답
수영하기 — attach 기 directly to the stem: 수영하 + 기 → 수영하기. This form is used in patterns like ~기 좋다 (good to), ~기 쉽다 (easy to), ~기 시작하다 (start to).
13.2read
Translate: 한국어 공부하기가 어려워요.
답
"Studying Korean is difficult." — 공부하기 is the nominalized verb acting as the sentence subject (~가 subject marker). ~기 is the natural choice when the nominalized verb is the subject or object of adjectives like 어렵다, 쉽다, 좋다.
13.3transform
Nominalize with ~는 것: 음악을 듣다 (listen to music → "listening to music / the act of listening to music")
답
음악을 듣는 것 — use the present modifier ~는 on the verb and follow with 것 (thing/fact): 듣 + 는 → 듣는. The full phrase 음악을 듣는 것을 좋아해요 (I like listening to music) is very common in speech.
13.4read
Translate: 영화 보는 것을 좋아해요.
답
"I like watching movies." — 영화 보는 것 = "the act of watching movies." ~는 것을 좋아하다 is the standard spoken construction for expressing what someone likes to do.
13.5transform
Nominalize with ~(으)ㅁ: 알다 (to know → formal noun form)
답
앎 — stem 알 ends in ㄹ; ㅁ is added alongside it, forming the double 받침 ㄻ: 알 + ㅁ → 앎. The ~(으)ㅁ nominalization is formal/written (e.g., notices, essays); everyday speech prefers ~기 or ~는 것.
13.6choose
Which nominalization is most natural here? "I started exercising." (Hint: ~기 시작하다 is the fixed pattern.)
  • 운동하기 시작했어요.
  • 운동하는 것 시작했어요.
답
운동하기 시작했어요 — the patterns ~기 시작하다 (start to), ~기 쉽다 (easy to), ~기 어렵다 (hard to), ~기 싫다 (dislike to) are fixed collocations that require ~기. ~는 것 does not slot into these patterns.
13.7choose
Which form fits better in a formal written notice? "Announcement: the meeting has been confirmed."
  • 회의가 확정됨을 알립니다.
  • 회의가 확정되는 것을 알립니다.
답
회의가 확정됨을 알립니다 — ~(으)ㅁ is the formal/written nominalization, natural for official notices and documents. ~는 것 is more casual and conversational. In written bureaucratic or academic Korean, ~(으)ㅁ is strongly preferred.
13.8identify
Three sentences use different nominalizations. Match each to its register: (A) 먹기가 좋아요. (B) 먹는 것이 좋아요. (C) 먹음이 확인됐다.
답
(A) ~기 — natural spoken; used in adjective patterns (~기 좋다). (B) ~는 것 — natural spoken/informal; "the fact/act of eating is good." (C) ~(으)ㅁ — formal/written; this phrasing (consumption was confirmed) would appear in reports or official language, not everyday speech.

14 Useful Sentence-Final Patterns

Endings that color the whole utterance: notice (네요), confirm (지요), realize (군요), you-know (잖아요), seems (것 같다), invite (ㄹ까요?), I'll (ㄹ게요), please (세요), want to (ㄹ래요?), try (아/어 보다). → read the rule.

14.1identify
What nuance does ~네요 add? 날씨가 좋네요! (The weather is nice!)
답
Mild surprise or fresh observation — the speaker is noticing or reacting to something in the moment. ~네요 implies "I'm just now realizing/observing this." It adds a tone of "oh, look at that!" rather than a neutral statement.
14.2identify
What nuance does ~지요? (~죠?) add? 맛있죠? (It's delicious, right?)
답
Seeking confirmation or shared agreement — the speaker believes the listener already knows or agrees with the proposition. ~지요 (contracted to ~죠 in speech) invites the listener to confirm: "You agree, right?"
14.3identify
What does ~군요 express? 그렇군요. (I see. / Oh, is that so.)
답
Realization or "ah, I see" — the speaker is registering newly understood information. ~군요 is slightly more formal; the casual equivalent is ~구나. Both signal that the speaker just learned or understood something, often in response to what was just said.
14.4identify
What does ~잖아요 convey? 그건 비싸잖아요. (That's expensive, you know.)
답
"As you know" / "you see" — the speaker reminds the listener of something both parties should already know. It has a slightly corrective or explanatory tone: the speaker is invoking shared knowledge to support their point.
14.5read
Translate and explain the pattern: 비가 오는 것 같아요.
답
"It seems to be raining." — ~(으)ㄴ/는 것 같다 expresses conjecture or a hedged opinion. Here 오는 is the present modifier (오 + 는) + 것 같아요. For past: 온 것 같아요 (seems it rained); for future/probability: 올 것 같아요 (seems it will rain).
14.6identify
What does ~(으)ㄹ까요? express? 같이 갈까요? (Shall we go together?)
답
Invitation or wondering — ~(으)ㄹ까요? is used to propose doing something together ("shall we?") or to express wondering aloud ("I wonder if…"). Stem 가 (vowel-final) + ㄹ까요 → 갈까요.
14.7identify
What does ~(으)ㄹ게요 express? 제가 할게요. (I'll do it.)
답
Speaker's promise or volitional commitment to the listener — ~(으)ㄹ게요 signals that the speaker is deciding or promising something for the listener's benefit. It can only have a first-person subject. Compare ~겠어요, which is a more general intention or supposition.
14.8choose
Which ending is correct for a speaker offering to carry the bag for someone?
  • 제가 들까요?
  • 제가 들게요.
답
제가 들게요 — ~(으)ㄹ게요 is the commitment form: "I'll do it (for you)." ~(으)ㄹ까요? is a question inviting the listener to decide ("shall I?"). If the speaker is volunteering unilaterally, ~(으)ㄹ게요 is correct. If genuinely asking whether to help, 들까요? would be appropriate.
14.9transform
Form the polite imperative with ~(으)세요: 앉다 (to sit → "please sit")
답
앉으세요 — stem 앉 ends in a consonant, so add the cushion 으: 앉 + 으세요 → 앉으세요. Compare vowel-final: 보다 → 보세요.
14.10transform
Form the "try doing" construction with ~아/어 보다: 입다 (to wear → "try wearing it")
답
입어 보세요 — stem 입: the vowel ㅣ is not ㅏ/ㅗ, so attach 어: 입 + 어 → 입어. Add 보세요 for the polite imperative of 보다 (try). The full construction is 입어 보다; 입어 봐요 is equally natural.
14.11identify
What does ~거든요 add? 못 가요. 시간이 없거든요. (I can't go. You see, I don't have time.)
답
Explaining with new information — ~거든요 introduces a reason or background fact that the listener does not yet know. It frames the preceding statement as needing justification. Contrast with ~잖아요, which invokes information the listener already knows.
14.12read
Identify the sentence-final pattern and its function: 같이 갈래요? (Do you want to go together?)
답
~(으)ㄹ래요? — "do you want to / will you?" Asks about the listener's desire or intention. Stem 가 + ㄹ래요 → 갈래요. Compare: 갈까요? proposes "shall we?" (joint decision); 갈래요? asks specifically about the listener's preference.

15 Quoting & Reported Speech

Direct (라고/이라고) vs indirect statement (다고), question (냐고), command (으라고), proposal (자고). → read the rule.

15.1identify
Is this direct or indirect speech? "가요"라고 했어요. (said "going")
답
Direct speech — the quoted words appear exactly as spoken, enclosed in quotation marks, followed by 라고 했어요. The 라고 directly follows the quoted word with no space. Compare indirect: 간다고 했어요 (said that he goes) — no quotation marks, verb form changes.
15.2transform
Convert to indirect statement: the person said 가요 (goes / is going). Use the indirect declarative form. (action verb, vowel-final stem)
답
간다고 했어요 — indirect statements for action verbs use ~ㄴ다고 (vowel-final stem) or ~는다고 (consonant-final stem). Stem 가 (vowel-final): 가 + ㄴ다고 → 간다고.
15.3transform
Convert to indirect statement: the person said 먹어요 (eats). Use the indirect declarative form. (action verb, consonant-final stem)
답
먹는다고 했어요 — consonant-final stem 먹 + 는다고. Note the contrast: vowel-final stems take ~ㄴ다고 (ㄴ fuses as final consonant); consonant-final stems take ~는다고.
15.4transform
Convert to indirect question: someone asked 가요? (Are you going?) (report: "asked whether [he] is going")
답
가냐고 물었어요 — indirect questions use ~냐고 for both action and descriptive verbs: stem 가 + 냐고. The reporting verb is typically 묻다/물어보다 (to ask), though 하다 is also used.
15.5transform
Convert to indirect command: someone told you 먹어요 (eat! / please eat). (report: "told [me] to eat")
답
먹으라고 했어요 — indirect commands use ~(으)라고: stem 먹 (consonant-final) + 으라고. Compare vowel-final: 가다 → 가라고 했어요 (no cushion needed). Do not confuse with the direct imperative ending.
15.6transform
Convert to indirect proposal: someone suggested 같이 가요 (let's go together). (report: "suggested going together")
답
같이 가자고 했어요 — indirect proposals use ~자고: attach 자고 to the bare stem: 가 + 자고 → 가자고. This applies to consonant-final stems too: 먹 + 자고 → 먹자고. No cushion, no vowel-harmony variation.
15.7choose
Which form correctly reports "He said it is a student" (noun + 이다 copula)?
  • 학생이라고 했어요.
  • 학생다고 했어요.
답
학생이라고 했어요 — the copula 이다 uses ~(이)라고 for indirect statements: after a consonant-final noun, 이라고; after a vowel-final noun, 라고 (e.g., 의사라고 했어요). The form ~다고 is for descriptive verb stems, not the copula.
15.8read
Identify the quote type in each sentence: (A) 간다고 했어요. (B) 가냐고 물었어요. (C) 가라고 했어요. (D) 가자고 했어요.
답
(A) ~ㄴ다고 — indirect statement (declarative): "said that (he) goes." (B) ~냐고 — indirect question: "asked whether (he) is going." (C) ~(으)라고 — indirect command: "told (him) to go." (D) ~자고 — indirect proposal: "suggested (they) go."

16 Numbers & Counters

Two number systems: Sino-Korean (일이삼…) for dates/money/minutes; native Korean (하나둘셋…) for ages/hours/most counted objects. → read the rule.

16.1write
Write today's date in Korean: 2026년 4월 26일. (spell out each number in Sino-Korean)
답
이천이십육 년 사월 이십육 일 — dates use Sino-Korean throughout: 2026 = 이천이십육, 4 = 사, 26 = 이십육. The markers are 년 / 월 / 일. Note: 4월 is 사월 (not 사 월 with a gap in speech).
16.2write
How do you say "10,000 won" and "5,000 won" in Korean? (money uses Sino-Korean)
답
만 원 (10,000 won) and 오천 원 (5,000 won). Money amounts always use Sino-Korean: 만 = 10,000; 오천 = 5,000. The currency word is 원 and it follows the number.
16.3write
Write "30 minutes" and "45 minutes" using Sino-Korean. (minutes use Sino-Korean + 분)
답
삼십 분 (30 minutes) and 사십오 분 (45 minutes). Minutes always take Sino-Korean: 삼십 = 30, 사십오 = 45. The counter is 분.
16.4read
Read this phone number aloud in Korean: 010-1234-5678. (phone numbers use Sino-Korean digits; 0 is 공)
답
공일공 — 일이삼사 — 오육칠팔. Phone numbers read each digit individually using Sino-Korean. Zero is 공 (not 영) in phone numbers. Each group is separated by a pause (represented by the dash).
16.5write
Write "20 years old" and "30 years old" using native Korean + the counter 살. (note the special pre-counter form at 20)
답
스무 살 (20 years old) and 서른 살 (30 years old). Age uses native Korean + 살. The number 20 has a special pre-counter form: 스물 → 스무 (not 스물 살). By contrast, 30 is 서른 with no change before 살.
16.6write
Write "3 o'clock" and "4 o'clock" using native Korean + 시. (hours use native Korean; 1–4 have special pre-counter forms)
답
세 시 (3 o'clock) and 네 시 (4 o'clock). Hours use native Korean with 시. The special pre-counter forms for 1–4 are: 하나 → 한, 둘 → 두, 셋 → 세, 넷 → 네. So you say 세 시, not 셋 시.
16.7write
Write "two people," "three books," and "five animals" using native Korean + the correct counter. (명 for people, 권 for books, 마리 for animals)
답
두 명 (two people), 세 권 (three books), 다섯 마리 (five animals). People: 명; books: 권; animals: 마리. Numbers 1–4 use special pre-counter forms (한, 두, 세, 네); 5 and above use the standard form (다섯).
16.8write
Write "one cup of coffee" and "four general items (things)" using native Korean + the correct counter. (잔 for cups/glasses, 개 for general things)
답
한 잔 (one cup) and 네 개 (four things). The pre-counter form of 하나 is 한, and of 넷 is 네. 잔 counts cups and glasses; 개 is the general-purpose counter for objects.
16.9write
Write "3:30" as a full Korean time expression. (hours = native, minutes = Sino-Korean)
답
세 시 삼십 분. Hours use native Korean: 세 시 (3 o'clock). Minutes use Sino-Korean: 삼십 분 (30 minutes). The two systems mix within a single time expression — this is expected and correct.
16.10choose
Which form correctly says "five years old"?
  • 오 살 (Sino-Korean + 살)
  • 다섯 살 (native Korean + 살)
답
다섯 살 — age uses native Korean + 살. 오 살 mixes Sino-Korean with the age counter and is incorrect. Native forms: 한 살 (1), 두 살 (2), 세 살 (3), 네 살 (4), 다섯 살 (5), …, 스무 살 (20), 서른 살 (30).

18 Question Words

누구/뭐/어디/언제/왜/어떻게/얼마/몇 — placed where the answer would go (no English-style fronting required). → read the rule.

18.1identify
What is the subject form of 누구, and why is it irregular? (who → who [as subject])
답
누가 — the subject form of 누구 is 누가 (not 누구가). This is a suppletive (irregular) form: the base changes. Compare regular nouns: 학생 + 이 → 학생이, 친구 + 가 → 친구가. Only 누구 behaves this way among the question words.
18.2write
Translate: "What is that?" (use 뭐 for conversational register)
답
저게 뭐예요? — 뭐 is the conversational form of 무엇 (what). The question word sits where the answer would go — here, in the predicate position: 저게 [뭐]예요?. Written/formal Korean uses 무엇이에요? or 무엇입니까? instead.
18.3write
Translate: "Where are you going?" (어디 + direction particle)
답
어디에 가요? — 어디 (where) takes the direction particle 에 when paired with a motion verb: 어디에 가다. In casual speech, 어디 가요? (particle dropped) is equally natural. The question word stays in its normal subject/object/adverbial position.
18.4transform
Turn this statement into a question using 언제: 내일 와요. (statement: "comes tomorrow" → question: "when does [he/she] come?")
답
언제 와요? — replace the time expression 내일 with 언제 (when) in the same position. Korean question words stand where the answer would be; no fronting to sentence-initial position is required (though moving to the front is also acceptable for emphasis).
18.5write
Translate: "Why did you come here?" (왜 — reason question)
답
왜 여기에 왔어요? — 왜 (why) typically appears at the front of the clause as an adverbial. The past tense of 오다 (to come) is 왔어요 (irregular: 오 + 았 → 왔).
18.6write
Translate: "How do you get to the station?" (어떻게 — manner/method)
답
역에 어떻게 가요? — 어떻게 (how) is the adverbial form of 어떻다. It asks about manner or method and typically precedes the verb. 역 = station; 에 marks the destination.
18.7write
Translate: "How much is this?" (얼마 — price/amount, no counter needed)
답
이게 얼마예요? — 얼마 (how much) asks about price or quantity and does not take a counter. It directly precedes the copula: 얼마예요? or 얼마입니까? (formal). Compare 몇, which always requires a counter: 몇 개예요?
18.8write
Translate: "How many people are there?" and "What time is it?" (몇 always needs a counter)
답
몇 명이에요? (how many people?) and 몇 시예요? (what time is it?). 몇 (how many/what number) always attaches to a counter: 몇 명 (people), 몇 개 (things), 몇 시 (o'clock), 몇 살 (years old). Never use 몇 alone without a counter.

19 Causative & Passive

Causative ~게 하다 (make X do); morphological passive ~이/히/리 (be done — 기 also exists, less frequent); ~아/어지다 (become / get). → read the rule.

19.1identify
What does ~게 하다 express, and is it lexical or productive? (example: 아이를 자게 했어요.)
답
~게 하다 is the productive (periphrastic) causative — it means "make/let someone do X" and can attach to any verb stem. 아이를 자게 했어요 = "I made the child sleep." The subject causes someone else (marked 을/를 or 에게) to perform the action. Because it's productive, it works with verbs that have no lexical causative form.
19.2transform
Make a causative sentence using ~게 하다: "I made my friend eat the food." (먹다 + 게 하다; friend = 친구)
답
친구에게 음식을 먹게 했어요. — attach 게 하다 to the bare stem: 먹 + 게 하다 → 먹게 하다. The person caused to act takes 에게 (or 를); the thing acted on takes 을/를. Past: 했어요.
19.3transform
Make a causative sentence using ~게 하다: "The teacher made the students study." (공부하다; students = 학생들)
답
선생님이 학생들을 공부하게 했어요. — 공부하다 → stem 공부하 + 게 하다 → 공부하게 하다. The students (학생들을) are the causee. Past tense: 공부하게 했어요.
19.4identify
Identify the passive form and its meaning: 산이 보여요. (보다 → passive)
답
보이다 — morphological passive of 보다 (to see) formed with ~이~: 보 + 이다 → 보이다 (to be seen / to be visible). 산이 보여요 = "The mountain is visible (can be seen)." The contracted polite form: 보이어요 → 보여요.
19.5identify
Identify the passive form and its meaning: 음악이 들려요. (듣다 → passive)
답
들리다 — morphological passive of 듣다 (to hear) formed with ~리~: stem 듣 → irregular ㄷ→ㄹ before 리: 들리다 (to be heard / to be audible). 음악이 들려요 = "The music is audible (can be heard)." Contracted: 들리어요 → 들려요.
19.6identify
Identify the passive forms and their base verbs: 문이 닫혔어요. / 문이 열렸어요. (the door was closed / the door was opened)
답
닫히다 (passive of 닫다, to close) and 열리다 (passive of 열다, to open). 닫다 + ~히~ → 닫히다; 열다 + ~리~ → 열리다. Past: 닫히 + 었어요 → 닫혔어요; 열리 + 었어요 → 열렸어요. The ㅣ+ㅓ contraction produces ㅕ in both cases.
19.7transform
Turn descriptive verbs into "become X" using ~아/어지다: 좋다 and 예쁘다. (좋다 = good; 예쁘다 = pretty)
답
좋아지다 (to become/get better) and 예뻐지다 (to become pretty / get prettier). 좋다: stem 좋, vowel ㅗ is bright → attach 아: 좋아지다. 예쁘다: stem 예쁘, ㅡ is neither ㅏ nor ㅗ → attach 어, but ㅡ+ㅓ contracts to ㅓ: 예쁘 + 어 → 예뻐, giving 예뻐지다.
19.8transform
Transform the active sentence into a passive one: 학생이 책을 읽어요. (The student reads the book. → The book is read by the student.)
답
책이 학생에게 읽혀요. — 읽다 (to read) → passive 읽히다 (to be read, suffix ~히~). In the passive: the object (책) becomes the subject + 이/가; the agent takes 에게: 학생에게. Polite present: 읽히 + 어요 → 읽혀요.

20 Irregular Verb Stems

Six irregular families: ㅂ / ㄷ / 르 / ㅡ / ㅅ / ㅎ — each transforms when meeting a vowel-starting ending. → read the rule.

20.1conjugate
Conjugate 춥다 to the polite present (해요체). (춥다 = to be cold)
답
추워요. — ㅂ-irregular: the final ㅂ changes to 우 before a vowel-starting ending. Stem 춥 + 어요 → 추우 + 어요 → 추워요. (Not 추버요 — the ㅂ never stays as a consonant before a vowel.)
20.2conjugate
Conjugate 무겁다 to the polite present (해요체), then to the polite past. (무겁다 = to be heavy)
답
Present: 무거워요. Past: 무거웠어요. — ㅂ-irregular: 무겁 → 무거우 before vowel. Present: 무거우 + 어요 → 무거워요. Past: 무거우 + 었어요 → 무거웠어요. The 우 + 어 contracts to 워.
20.3conjugate
Conjugate 듣다 and 걷다 to the polite present. (듣다 = to listen; 걷다 = to walk)
답
들어요 / 걸어요. — ㄷ-irregular: the final ㄷ changes to ㄹ before a vowel-starting ending. 듣 + 어요 → 들어요; 걷 + 어요 → 걸어요. Contrast regular 받다 → 받아요 (the ㄷ stays) — irregularity is per-verb.
20.4conjugate
Conjugate 묻다 (to ask) to the polite present. (Be careful — this 묻다 means "ask," not "bury." Same spelling, different irregularity pattern.)
답
물어요. — 묻다 (to ask) is a ㄷ-irregular: 묻 + 어요 → 물어요. Note: 묻다 meaning "to bury" is a regular verb — 묻어요. Same spelling, different lexical item, different irregularity. The "ask" sense is irregular; the "bury" sense is regular.
20.5conjugate
Conjugate 빠르다 and 모르다 to the polite present. (빠르다 = to be fast; 모르다 = to not know)
답
빨라요 / 몰라요. — 르-irregular: the 으 of 르 drops, and the ㄹ doubles onto the preceding syllable. Then vowel harmony applies: 빠르 — preceding vowel ㅏ is bright → 빨 + 라요 → 빨라요. 모르 — preceding vowel ㅗ is bright → 몰 + 라요 → 몰라요.
20.6conjugate
Conjugate 부르다 to the polite present and polite past. (부르다 = to call / to sing)
답
Present: 불러요. Past: 불렀어요. — 르-irregular: preceding vowel of 부르 is ㅜ, which is neither ㅏ nor ㅗ, so dark → attach 러요: 불 + 러요 → 불러요. Past: 불 + 렀어요 → 불렀어요.
20.7conjugate
Conjugate 크다 and 바쁘다 to the polite present. (크다 = to be big; 바쁘다 = to be busy)
답
커요 / 바빠요. — ㅡ-irregular: ㅡ drops before 아/어, and vowel harmony is determined by the syllable before the dropped ㅡ. 크다: no preceding syllable → defaults to dark → 크 + 어요 → 커요. 바쁘다: preceding syllable 바 has bright ㅏ → 바쁘 + 아요 → 바빠요.
20.8conjugate
Conjugate 짓다 (to build) and 낫다 (to heal / be better) to the polite present.
답
지어요 / 나아요. — ㅅ-irregular: the final ㅅ drops before a vowel-starting ending, but the vowel of the ending remains. 짓 + 어요 → 지어요 (not 짓어요 or 져요). 낫 + 아요 → 나아요 (the ㅅ drops; 아 stays — not 나요). Contrast regular 웃다 → 웃어요.
20.9conjugate
Conjugate 그렇다 (to be so/that way) and 빨갛다 (to be red) to the polite present.
답
그래요 / 빨개요. — ㅎ-irregular (descriptive verbs only): ㅎ drops and the preceding vowel changes. 그렇: ㅎ drops, ㅓ + ㅏ/어 → ㅐ: 그렇 + 어요 → 그래요. 빨갛: ㅎ drops, ㅏ + ㅏ/어 → ㅐ: 빨갛 + 아요 → 빨개요. Never write 그러요 or 빨가요.
20.10conjugate
Conjugate 쉽다 (to be easy) to polite present, then explain the vowel harmony choice.
답
쉬워요. — ㅂ-irregular: 쉽 + vowel-starting ending → 쉽 changes to 쉬우. The stem vowel is ㅣ (in 쉬), which is neither ㅏ nor ㅗ → dark ending 어요. But once 우 is added, the combined 우 + 어 contracts to 워: 쉬우 + 어요 → 쉬워요.
20.11identify
Identify the irregular family for each verb: 걷다, 부르다, 낫다, 빨갛다, 바쁘다, 춥다.
답
걷다 → ㄷ-irregular (ㄷ→ㄹ before vowel). 부르다 → 르-irregular (double ㄹ). 낫다 → ㅅ-irregular (ㅅ drops). 빨갛다 → ㅎ-irregular (ㅎ drops, vowel changes). 바쁘다 → ㅡ-irregular (ㅡ drops, harmony from previous syllable). 춥다 → ㅂ-irregular (ㅂ→우 before vowel).
20.12identify
Two verbs look like they might be irregular — but one is regular. Which is irregular and which is regular? Verbs: 받다 (to receive) vs. 듣다 (to listen)
답
받다 is regular: 받 + 아요 → 받아요 (the ㄷ stays as a coda before the vowel). 듣다 is ㄷ-irregular: 듣 + 어요 → 들어요 (the ㄷ changes to ㄹ). Irregularity is a per-verb property — all ㄷ-final verbs appear similar but only some are irregular. You must memorize which ones change.

21 Building a Sentence — synthesis drills

Assemble a complete Korean sentence end-to-end: subject → particles → verb → tense → speech level → final particle. → read the rule.

21.1build
Build a sentence: "I'm going to school" — 해요체, present tense, with topic marker.
답
저는 학교에 가요. — Subject + topic 는 + destination 에 + verb 가다 → 가요.
21.2build
Build a sentence: "I'm reading a book" — 해요체, present tense, topic marker + direct object marker.
답
저는 책을 읽어요. — Subject 저 + topic 는 → 저는. Object 책 + direct-object marker 을 → 책을. Verb 읽다: stem 읽, vowel ㅣ is not ㅏ/ㅗ → dark → 읽 + 어요 → 읽어요.
21.3build
Build a sentence: "I went to school" — 해요체, past tense.
답
저는 학교에 갔어요. — Same structure as 21.1 but past tense. Verb 가다: stem 가, vowel ㅏ is bright → 가 + 았어요 → 갔어요 (contraction: ㅏ + ㅏ → ㅏ, ㅅ adds as coda).
21.4build
Build a sentence with honorifics: "The teacher reads a book" — 해요체, present, with ~(으)시~ honorific for the subject.
답
선생님은 책을 읽으세요. — Subject 선생님 takes topic 은 (ends in consonant). Verb 읽다: stem 읽 ends in consonant → honorific ~으시~: 읽 + 으시 + 어요 → 읽으세요 (the 시 + 어 contracts to 세).
21.5build
Build a sentence with a connector clause: "I went to school and (then) met my friend" — 해요체, past tense, use ~아/어서 connector.
답
저는 학교에 가서 친구를 만났어요. — Connector ~아/어서 links sequential events. 가다: bright ㅏ → 가 + 아서 → 가서 (contracted). The tense marker goes only on the final verb: 만나다 + 았어요 → 만났어요. Clause order: first event → second event (verb last).
21.6build
Build a question using a question word: "Who went to school yesterday?" — 해요체, past tense. (어제 = yesterday; 누가 = who + subject marker)
답
어제 누가 학교에 갔어요? — Time word 어제 moves to the front. The question word 누가 (who + subject marker 이/가) stays in subject position. Verb stays last: 가다 → past 갔어요. The rising intonation at the end (or question mark in writing) signals the question — no separate question particle needed in 해요체.
21.7build
Build a sentence using a sentence-final exclamatory ending: "Oh, the weather is nice!" — use ~네요 (observation/mild surprise).
답
날씨가 좋네요! — Subject 날씨 (weather) + subject marker 가. Descriptive verb 좋다: attach ~네요 directly to the stem: 좋 + 네요 → 좋네요. The ending ~네요 signals that the speaker is noticing or mildly surprised by something they just perceived — appropriate for reacting to the weather in the moment.
21.8build
Full synthesis — build a sentence: "Tomorrow I'd like to go to a cafe with a friend to drink coffee" — 해요체, use ~(으)러 (purpose of motion) + ~고 싶다 (want to).
답
내일 친구하고 카페에서 커피를 마시러 가고 싶어요. — Recipe: Time (내일) → with-companion (친구하고) → location (카페에서) → object (커피를) → purpose clause (마시러: stem 마시 + ~러) → motion verb (가다) + want (~고 싶다) → polite present: 가고 싶어요. Verb last; connectors attach in order of logical dependency.

23 Identify Endings

Twelve common sentence-final patterns from §14/§23 — for each, name what the ending signals. → read the rule.

23.1identify
What does ~네요 signal? Identify and explain in context: 비가 오네요.
답
~네요 signals fresh observation / mild surprise. The speaker is noticing something in the moment — reacting to new information they just perceived. 비가 오네요 = "Oh, it's raining (I just noticed)." Attach directly to the verb stem: 오 + 네요 → 오네요. Use it when you just saw, heard, or realized something — not for pre-known facts.
23.2identify
What does ~지요 / 죠 signal? Identify and explain in context: 맛있죠?
답
~지요 (contracted to ~죠) signals confirmation-seeking / shared assumption. The speaker assumes the listener agrees and invites confirmation. 맛있죠? = "It's delicious, right?" / "You agree it's good, don't you?" Attach to the verb stem: 맛있 + 지요 → 맛있죠. Softer and more conversational than a plain question.
23.3identify
What does ~군요 signal? Identify and explain in context: 아, 그렇군요.
답
~군요 signals realization / dawning understanding. The speaker is processing new information they just learned (often verbally, from what someone said). 아, 그렇군요 = "Ah, I see / Oh, is that so!" Attach to the verb stem: 그렇 + 군요 → 그렇군요. Compare with ~네요 (perception-based) — ~군요 is more about receiving new information from speech.
23.4identify
What does ~잖아요 signal? Identify and explain in context: 어제 만났잖아요.
답
~잖아요 signals reminding the listener of something they already know ("as you know / you know that…"). 어제 만났잖아요 = "We met yesterday, you know." Attach to the past form: 만났 + 잖아요 → 만났잖아요. Often used when the speaker feels the listener should remember or acknowledge something. Can carry a mild reproachful or assertive nuance.
23.5identify
What does ~거든요 signal? Identify and explain in context: 시간이 없거든요.
답
~거든요 signals providing a reason or background information the listener doesn't know yet ("you see… / it's because…"). 시간이 없거든요 = "It's because I don't have time, you see." Attach to the verb stem: 없 + 거든요 → 없거든요. Often used as a follow-up explanation or as a soft refusal. The information is new to the listener — unlike ~잖아요 which recalls shared knowledge.
23.6identify
What does ~(으)ㄴ/는데 signal? Identify and explain in context: 비가 오는데 우산이 없어요.
답
~(으)ㄴ/는데 signals background / contrast / soft "but". The first clause sets up context or background; the second clause provides the main point or contrasting fact. 비가 오는데 우산이 없어요 = "It's raining (background), but I don't have an umbrella." Action verb present: 오 + 는데 → 오는데. The ending doesn't assert cause-effect — it just frames the two clauses against each other.
23.7identify
What does ~(으)면서 signal? Identify and explain in context: 음악을 들으면서 공부해요.
답
~(으)면서 signals simultaneous actions by the same subject ("while doing X, [also] do Y"). 음악을 들으면서 공부해요 = "I study while listening to music." Stem 듣 (ㄷ-irregular → 들) + 으면서 → 들으면서. Both actions happen at the same time, performed by the same subject. Do not use ~면서 when the two subjects are different.
23.8identify
What does ~(으)려고 signal? Identify and explain in context: 사려고 가게에 갔어요.
답
~(으)려고 signals intention / purpose ("intending to / in order to"). 사려고 가게에 갔어요 = "I went to the store intending to buy (something)." Stem 사 (no final consonant) + 려고 → 사려고. The two verbs share the same subject. Unlike ~(으)러 (which only occurs with motion verbs), ~(으)려고 is not restricted to motion verbs.
23.9identify
What does ~(으)면 안 되다 signal? Identify and explain in context: 늦으면 안 돼요.
답
~(으)면 안 되다 signals prohibition / must not ("it won't do if… / you must not"). 늦으면 안 돼요 = "You must not be late / It won't do if you're late." Stem 늦 ends in consonant → 으면: 늦 + 으면 → 늦으면. Literally: "if [X] happens, it's not okay." Contrast with ~아/어도 되다 = "it's okay to / may": 늦어도 돼요 = "it's okay to be late."
23.10identify
What does ~아/어야 되다 signal? Identify and explain in context: 가야 돼요.
답
~아/어야 되다 signals obligation / must do ("have to / need to"). 가야 돼요 = "I have to go / I must go." Stem 가 + bright vowel ㅏ → 가 + 아야 → 가야 (contracted) + 되다 → 돼요. The variant ~아/어야 하다 is slightly more formal but identical in meaning: 가야 해요.
23.11identify
What does ~(으)ㄹ 수 있다/없다 signal? Identify and explain in context: 갈 수 있어요. / 갈 수 없어요.
답
~(으)ㄹ 수 있다 = can / is able to; ~(으)ㄹ 수 없다 = cannot / is unable to. 갈 수 있어요 = "I can go." 갈 수 없어요 = "I can't go." Stem 가 (no final consonant) → ㄹ: 가 + ㄹ → 갈. The 수 is a dependent noun ("way / means"); 있다/없다 asserts existence or absence of that ability. Tense goes on the final 있다/없다: 있었어요 / 없었어요 for past ability.
23.12identify
What does ~아/어 보다 signal? Identify and explain in context: 한국어를 공부해 봐요.
답
~아/어 보다 signals trying / attempting an action ("try doing / give X a try"). 한국어를 공부해 봐요 = "Try studying Korean." 공부하다: the 하 + 여 → 해, then + 보다 → 해 보다 → 해 봐요 (contracted polite). The auxiliary 보다 (literally "to see/look") implies making an attempt to find out what something is like. Also used in past: 먹어 봤어요 = "I tried eating it."
