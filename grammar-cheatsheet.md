# Korean Grammar Cheat Sheet

## 1. Word Order — SOV

Korean is **Subject – Object – Verb**. The verb always ends the sentence.

```
저는    사과를    먹어요.
[주어]  [목적어]   [동사]
나      apple      eat
```

- Modifiers come **before** what they modify (adj before noun, adv before verb).
- Particles attach to the noun/phrase, not the word that follows.
- Subjects and objects are routinely dropped when context is clear.

---

## 2. Particles (조사)

Particles glue grammar to nouns. They are chosen based on whether the noun ends in a **vowel** or a **consonant (받침)**.

### Topic — 은 / 는
- Marks the topic ("as for X").
- 받침 → 은, no 받침 → 는.
- Used for contrast and to introduce new topics.
```
저는 학생이에요.   (As for me, I'm a student.)
김치는 매워요.    (Kimchi, [it] is spicy.)
```

### Subject — 이 / 가
- Marks the grammatical subject; emphasizes "who/what" did it.
- 받침 → 이, no 받침 → 가.
```
누가 왔어요?   —   민수가 왔어요.
```

**은/는 vs 이/가:** 은/는 = topic / contrast / general statement. 이/가 = new info / specific identification / answers "who?".

### Object — 을 / 를
- Marks the direct object.
- 받침 → 을, no 받침 → 를.
```
책을 읽어요.    물을 마셔요.    영화를 봐요.
```

### Possessive / "of" — 의
```
저의 책 → 제 책   (my book)
친구의 집           (friend's house)
```

### Location / Time — 에
- Static location, destination, time.
```
학교에 가요.   (going to school)
3시에 만나요. (meet at 3)
집에 있어요.  (am at home)
```

### Action location — 에서
- Where an action happens, or "from".
```
도서관에서 공부해요.   (study at the library)
한국에서 왔어요.       (came from Korea)
```

### To a person — 에게 / 한테 / 께
- 에게 (written), 한테 (spoken), 께 (honorific).
```
친구에게 선물을 줬어요.
선생님께 말씀드렸어요.
```

### From a person — 에게서 / 한테서
```
엄마한테서 편지를 받았어요.
```

### With / and — 와/과, 하고, (이)랑
- 와/과 (formal): 받침 → 과, no 받침 → 와.
- 하고 (neutral), (이)랑 (casual).
```
빵과 우유   /   빵하고 우유   /   빵이랑 우유
```

### Also / too — 도
- Replaces 은/는/이/가/을/를. Never combine with them.
```
저도 가요.   (I'm going too.)
```

### Only — 만
```
물만 마셔요.   (I drink only water.)
```

### From ~ to — 에서 ~ 까지 (place), 부터 ~ 까지 (time)
```
서울에서 부산까지
9시부터 6시까지
```

### By means of / with — (으)로
- 받침 → 으로, no 받침/ㄹ → 로. Tool, method, direction.
```
젓가락으로 먹어요.   (eat with chopsticks)
한국어로 말해요.    (speak in Korean)
학교로 가요.        (head toward school)
```

## 3. The Copula — 이다 / 아니다

- **이다** = "to be (X)".  No 받침 → ~예요 / 받침 → ~이에요.
- **아니다** = "to not be (X)".  Noun takes 이/가.
```
저는 의사예요.            (I am a doctor.)
저는 학생이에요.          (I am a student.)
저는 학생이 아니에요.     (I am not a student.)
```

존재 (existence): **있다 / 없다** = "there is / isn't", "have / don't have".
```
시간이 있어요.   돈이 없어요.
```

---

## 4. Verb / Adjective Stems

Drop **다** from the dictionary form to get the stem.

```
가다 → 가-     먹다 → 먹-     예쁘다 → 예쁘-
```

Korean **adjectives are descriptive verbs** — they conjugate just like verbs, no copula needed.
```
하늘이 파래요.   (The sky is blue.)   ← 파랗다 conjugated directly
```

## 5. Speech Levels (politeness)

| Level | Ending | When |
|---|---|---|
| 하십시오체 (formal-polite) | ~(스)ㅂ니다 / ~(스)ㅂ니까? | News, business, military, strangers |
| 해요체 (polite) | ~아요/어요/해요 | Default everyday polite |
| 해체 / 반말 (casual) | ~아/어/해 | Friends, family, kids |
| 한다체 (plain/written) | ~ㄴ/는다 | Books, news, diaries |

You'll mostly live in **해요체** and **반말**.

---

## 6. Present Tense — 아/어/해요

Pick by the stem's last vowel.

| Stem vowel | Ending | Example |
|---|---|---|
| ㅏ, ㅗ | ~아요 | 살다 → 살아요 |
| Other vowels | ~어요 | 먹다 → 먹어요 |
| 하- verbs | 해요 | 공부하다 → 공부해요 |

### Vowel contractions (when stem ends in a vowel)
```
가 + 아요 → 가요          오 + 아요 → 와요
보 + 아요 → 봐요          주 + 어요 → 줘요
마시 + 어요 → 마셔요      서 + 어요 → 서요
배우 + 어요 → 배워요      되 + 어요 → 돼요
```

For formal: stem + ㅂ니다 (vowel-ending) / 습니다 (consonant-ending).
```
가다 → 갑니다     먹다 → 먹습니다
```

---

## 7. Past Tense — 았/었/했어요

Same vowel rule, plus 었.
```
가다 → 갔어요         먹다 → 먹었어요
하다 → 했어요         보다 → 봤어요
공부하다 → 공부했어요
```
Formal past: ~았/었습니다.

---

## 8. Future / Probability

### ~(으)ㄹ 거예요 — plain future / guess
- 받침 → 을 거예요, no 받침 → ㄹ 거예요.
```
내일 갈 거예요.       (I will go tomorrow.)
비가 올 거예요.       (It will probably rain.)
```

### ~겠~ — intention / strong assumption / politeness
```
제가 하겠습니다.       (I will do it.)
맛있겠어요!            (Looks delicious!)
알겠어요.              (Got it.)
```

### ~(으)려고 하다 — plan to
```
한국에 가려고 해요.    (I'm planning to go to Korea.)
```

---

## 9. Negation

| Type | Pattern | Example |
|---|---|---|
| Short | 안 + verb | 안 먹어요 |
| Long | verb stem + 지 않다 | 먹지 않아요 |
| Inability (short) | 못 + verb | 못 가요 |
| Inability (long) | stem + 지 못하다 | 가지 못해요 |
| Don't (imperative) | stem + 지 마세요 | 가지 마세요 |

Note: 하다 verbs split — **공부 안 해요** (not 안 공부해요).

---

## 10. Honorifics — ~(으)시~

Insert **시** into the verb stem to honor the **subject**.
```
가다 → 가시다 → 가세요 / 가십니다
먹다 → 드시다 (special)
자다 → 주무시다 (special)
있다 → 계시다 (special, "to be present")
```
Honorific particles: 께서 (subject), 께 (to). Honorific noun forms: 진지 (밥), 댁 (집), 연세 (나이), 성함 (이름), 말씀 (말).

Humble forms (lower yourself): 드리다 (주다), 뵙다 (보다), 여쭙다 (묻다).

---

## 11. Connectors (clause linking)

| Meaning | Pattern | Example |
|---|---|---|
| and (then) | ~고 | 먹고 자요. |
| and (sequence, same subject) | ~아/어서 | 학교에 가서 공부해요. |
| because (reason) | ~아/어서 | 추워서 안 가요. |
| because (subjective, opinion/excuse) | ~(으)니까 | 비가 오니까 우산 가져가요. |
| but | ~지만 | 좋지만 비싸요. |
| but / however | ~는데 / ~(으)ㄴ데 | 먹었는데 배고파요. |
| if | ~(으)면 | 시간 있으면 만나요. |
| while / as | ~(으)면서 | 음악 들으면서 공부해요. |
| in order to | ~(으)러 (with go/come) | 밥 먹으러 가요. |
| so that / for | ~기 위해서 | 건강을 위해서 운동해요. |
| even though | ~아/어도 | 비싸도 살 거예요. |
| or | ~거나 / 아니면 | 영화 보거나 책 읽어요. |

**~서 vs ~(으)니까:** 서 = objective cause; cannot be used with imperatives/suggestions. 니까 = subjective; required with 명령/청유 ("let's", "do this").

**~는데:** background/setup before the main clause — extremely common, very versatile (contrast, context, soft "but").

---

## 12. Modifiers (turning verbs/adj into noun-modifiers)

Korean has no relative pronouns. The modifier sits **before** the noun.

| Type | Past | Present | Future |
|---|---|---|---|
| Action verb | ~(으)ㄴ | ~는 | ~(으)ㄹ |
| Descriptive (adj) | — | ~(으)ㄴ | ~(으)ㄹ |
| 있다/없다 | — | ~는 | ~(으)ㄹ |

```
어제 본 영화        (the movie I saw yesterday)
지금 보는 영화      (the movie I'm watching now)
내일 볼 영화        (the movie I'll watch tomorrow)
예쁜 사람           (a pretty person)
재미있는 책         (an interesting book)
```

---

## 13. Nominalization (verb → noun)

| Suffix | Use |
|---|---|
| ~기 | Generic noun ("the act of"); used in patterns like ~기 쉽다, ~기 시작하다 |
| ~(으)ㅁ | More abstract / formal noun; sign-style writing (e.g. 모임, 믿음) |
| ~는 것 | "the thing of ~ing"; common in spoken explanations |

```
한국어 공부하기가 어려워요.
영화 보는 것을 좋아해요.
```

## 14. Useful sentence-final patterns

| Pattern | Meaning |
|---|---|
| ~고 싶다 | want to |
| ~아/어 보다 | try -ing |
| ~아/어 주다 | do for someone |
| ~아/어야 하다/되다 | have to / must |
| ~아/어도 되다 | may, it's OK to |
| ~(으)면 안 되다 | must not |
| ~(으)ㄹ 수 있다/없다 | can / cannot |
| ~(으)ㄹ 줄 알다/모르다 | know / don't know how to |
| ~(으)ㄴ/는 것 같다 | seems / I think |
| ~(으)ㄹ 것 같다 | will probably / seems will |
| ~(으)ㄹ게요 | I'll (promise / commitment to listener) |
| ~(으)ㄹ래요? | will you / want to? |
| ~지요? (~죠?) | right?, isn't it? (seeking agreement) |
| ~네요 | mild surprise / observation |
| ~군요 / ~구나 | realization ("ah, I see") |
| ~잖아요 | as you know / you see |
| ~거든요 | you see (giving reason) |
| ~(으)ㄴ/는데요 | softening / leading remark |

## 15. Quoting & reported speech

Direct: ~(이)라고 하다.
```
"가요" 라고 했어요.
```

Indirect (contractions are normal in speech):

| Type | Statement | Question | Suggestion | Command |
|---|---|---|---|---|
| Verb | ~(느)ㄴ다고 | ~(느)냐고 | ~자고 | ~(으)라고 |
| Adjective | ~다고 | ~(으)냐고 | — | — |
| 이다 | ~(이)라고 | ~(이)냐고 | — | — |

```
간다고 했어요.        (said [he] goes)
가냐고 물었어요.      (asked if [he] goes)
가자고 했어요.        (said let's go)
가라고 했어요.        (told [him] to go)
```

Common contractions: ~다고 해요 → ~대요, ~(이)라고 해요 → ~(이)래요, ~냐고 해요 → ~냬요, ~자고 해요 → ~재요, ~(으)라고 해요 → ~(으)래요.

## 16. Numbers & counters

Korean has **two** number systems. Pick by what's being counted.

| Sino-Korean (일,이,삼…) | Native (하나, 둘, 셋…) |
|---|---|
| Dates, money, minutes, phone, math, floors, addresses | Hours, age, people, things, animals, drinks |

### Native form changes before counters
```
하나 → 한,   둘 → 두,   셋 → 세,   넷 → 네,   스물 → 스무
```

### Common counters
| Counter | For |
|---|---|
| 개 | things (general) |
| 명 / 사람 / 분(honorific) | people |
| 마리 | animals |
| 권 | books |
| 잔 | cups/glasses |
| 병 | bottles |
| 장 | sheets / tickets |
| 대 | machines / cars |
| 살 | years of age |
| 시 / 시간 | o'clock / hours |
| 번 | times (frequency / number) |

```
사과 세 개       커피 두 잔       스무 살       3시 30분
```

### Time
- Hours = native + 시. Minutes = sino + 분.
```
일곱 시 십오 분 = 7:15
```

### Dates
```
2026년 4월 25일   (sino throughout)
```

## 17. Pronouns (skip them when context allows)

| Person | Plain | Humble |
|---|---|---|
| I | 나 | 저 |
| my | 내 | 제 |
| we | 우리 | 저희 |
| you (sing.) | 너 | (avoid; use name + 씨 / title) |
| he/she/they | 그 / 그녀 / 그들 (mostly literary) |

Calling someone "you" directly is often rude. Use **이름 + 씨**, **선생님**, **사장님**, family terms (오빠, 언니, 누나, 형, 아저씨, 아주머니) instead.

## 18. Question words

| 누구 | who | 무엇 / 뭐 | what |
|---|---|---|---|
| 언제 | when | 어디 | where |
| 왜 | why | 어떻게 | how |
| 얼마 | how much | 몇 | how many (+ counter) |
| 어느 | which | 어떤 | what kind |

Yes/no questions just rise in tone in 해요체. Formal uses ~ㅂ니까/습니까?.

---

## 19. Causative & Passive

These often use built-in derived forms — memorize per verb.

- **Causative** (~게 하다 is the productive form):
```
아이를 자게 했어요.   (I made the child sleep.)
```
- **Passive** (~아/어지다 is productive; many verbs have lexical passives):
```
문이 열렸어요.   (The door was opened.)   ← from 열다
컵이 깨졌어요.   (The cup broke.)         ← 깨다 → 깨지다
```

---

## 20. Irregular verb stems

| Type | Trigger | Example |
|---|---|---|
| ㅂ-irreg | before vowel: ㅂ → 우 | 춥다 → 추워요 |
| ㄷ-irreg | before vowel: ㄷ → ㄹ | 듣다 → 들어요 |
| ㅅ-irreg | before vowel: ㅅ drops | 짓다 → 지어요 |
| ㄹ-stem | drops ㄹ before ㄴ/ㅂ/ㅅ | 살다 → 삽니다, 사세요 |
| 으-irreg | 으 drops before 아/어 | 예쁘다 → 예뻐요, 쓰다 → 써요 |
| 르-irreg | 르 → ㄹ라/ㄹ러 + double ㄹ | 모르다 → 몰라요, 부르다 → 불러요 |
| ㅎ-irreg (adj) | ㅎ drops, vowel changes | 빨갛다 → 빨개요, 어떻다 → 어때요 |
| 하다 | 하 + 여 → 해 | 하다 → 해요 |

Regular ㅂ/ㄷ/ㅅ verbs exist (입다 → 입어요, 받다 → 받아요, 웃다 → 웃어요) — irregularity is per-verb, must be learned.

## 21. Building a sentence — the recipe

```
[Time]  [Topic은/는]  [Subject이/가]  [Place에/에서]  [Indirect obj에게]  [Object을/를]  [Adverb]  [Verb]
어제    저는                          학교에서                              친구를      오래       기다렸어요.
```

- Time often goes first; topic before subject; verb always last.
- Drop anything obvious from context.

---

## 22. Quick decision tree for endings

1. **Who am I talking to?** → pick speech level (해요 vs 합니다 vs 반말).
2. **When?** → past 았/었, present 아/어, future ~(으)ㄹ 거예요 / ~겠.
3. **Honor the subject?** → insert ~(으)시~.
4. **What's the stem's last vowel?** → choose 아 vs 어.
5. **Does the stem end in a 받침?** → choose 은/는, 이/가, 을/를, 으- vs no-으-.
6. **Is the verb irregular?** → apply the irregular rule.
7. **Connecting clauses?** → pick the connector by nuance (서 vs 니까, 지만 vs 는데).
8. **Modifying a noun?** → pick the modifier ending by tense + verb type.

## 24. Quick study tips

- **Learn 받침-vs-vowel pairs together** (은/는, 이/가, 을/를, (으)면, (으)니까…). The pattern is identical, just one rule.
- **Don't translate "to be"** — sometimes 이다, sometimes 있다, often nothing (adjectives conjugate themselves).
- **Particles ≠ word order**, but particles + word order together carry meaning. Don't drop both.
- **Listen for sentence endings** — politeness, mood, evidentiality all live there.
- **Embrace 는데** — it's the duct tape of Korean conversation.
